Aplikasi ini digunakan untuk mereka yang lupa password login admin aplikasi iPos4, iPos5, Apotek & Klinik v4 ataupun Resto v3
Cara reset password iPos cukup menjalankan aplikasi ini, tekan tombol Load Database, pilih nama Database nya lalu tekan tombol Reset.
Jika berhasil direset, maka password login user admin akan kembali menjadi default: admin kembali.
